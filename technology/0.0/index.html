Advanced: @import & Third-Party Injection

Apa itu @import?

@import url('...') memuat stylesheet eksternal ke dalam kode kamu. Creator Ko-Fi menggunakannya supaya pembeli cukup menempelkan satu baris untuk desain kompleks.

@import url('https://raw.githubusercontent.com/
  username/repo/main/style.css');

/* File style.css HARUS menggunakan
  class pp- juga, bukan .css-XXXX */

Kenapa GitHub Raw / Stylus?

GitHub Raw URL mengirim CSS murni tanpa HTML wrapper. Stylus adalah ekstensi browser penyimpan CSS kustom. Keduanya dipakai karena mengirim file text plain.

raw.githubusercontent.com gist.github.com pastebin.com/raw

Sanitasi & Specificity

Janitor AI menyaring CSS yang dianggap berbahaya. Mereka juga membungkus CSS bawaan dengan specificity tinggi (ID selector). Gunakan !important di setiap deklarasi.

Wajib: Selalu gunakan !important pada setiap properti. Tanpa itu, CSS bawaan Janitor akan menang.

@import Font Eksternal

@import untuk Google Fonts biasanya tidak diblokir CSP. Ini trik paling aman dan reliable untuk memuat font kustom.

@import url('https://fonts.googleapis.com/
  css2?family=Orbitron:wght@400;700');

.pp-uc-background h2 {
  font-family: 'Orbitron', sans-serif
    !important;
}
